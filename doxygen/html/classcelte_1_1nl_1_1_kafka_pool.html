◆ CatchUp()

void celte::nl::KafkaPool::CatchUp ( unsigned int maxBlockingMs = 10)

Executes all the callbacks associated with all the messages received since the last time this method was called.

This method is blocking!

If there are too many tasks to complete, this method will not take up any new task after maxBlockingMs milliseconds.
