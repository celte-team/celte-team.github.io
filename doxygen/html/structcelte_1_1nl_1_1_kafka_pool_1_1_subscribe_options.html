Detailed Description

Options for subscribing to a Kafka topic.

This structure holds various options that can be configured when subscribing to a Kafka topic.

Member Data Documentation

◆ autoCreateTopic

bool celte::nl::KafkaPool::SubscribeOptions::autoCreateTopic = true

Flag indicating whether the topic should be automatically created if it does not exist. Default is true.

◆ autoPoll

bool celte::nl::KafkaPool::SubscribeOptions::autoPoll = true

Flag indicating whether messages should be automatically polled. Default is true. If set to false, the user must manually call the Poll method to receive messages. The user should wait for a few ms before calling Poll as the creation of a consumer takes some time to process in the kafka cluster.
